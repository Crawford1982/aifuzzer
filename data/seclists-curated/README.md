# Curated ID/path token slice (not the full SecLists repo)

Small, **reviewed** values for ID-like path fuzzing. Mythos loads this only when you pass **`--curated-wordlist`** (still subject to **`--max-wordlist-injections`** and scope/RPS).

To use full SecLists or custom lists, point **`--wordlist`** at your own file instead.
